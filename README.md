# Nuxion

## Features
- [X] List of notes
- [X] Notes CRUD
- [X] Note ID slugs 
- [X] Use localStorage

## Additional feature
- [ ] Trash bin
- [ ] Use Firebase (or alt) for auth and storage
- [ ] WYSIWYG editor

## Setup

Make sure to install the dependencies:

```bash
# npm
npm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev
```
